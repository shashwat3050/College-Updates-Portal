# Django News Portal

A Django-based web application built and deployed from scratch, covering the complete development-to-deployment workflow.

## Features
- User-friendly news listing interface
- Django Admin for content management
- Responsive UI (mobile + desktop)
- Production deployment on Railway
- Secure configuration with CSRF and ALLOWED_HOSTS
- Static file handling using WhiteNoise

## Tech Stack
- Python
- Django
- HTML, CSS
- Gunicorn
- Railway

## Deployment
The application is deployed on Railway using Gunicorn as the WSGI server.

## Learning Outcome
This project helped me understand real-world Django deployment challenges, including static file handling, security configurations, and production debugging.
